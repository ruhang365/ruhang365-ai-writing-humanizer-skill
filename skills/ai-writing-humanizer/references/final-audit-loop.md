# Final Audit Loop

Use this after the publishable draft is written. The goal is not to rewrite again from scratch. The goal is to catch the last obvious AI traces.

## Loop

1. Read the rewritten draft without looking at the original prompt.
2. Ask: "Where does this still obviously feel like AI?"
3. List only high-impact residual issues.
4. Fix those issues with the smallest edit that works.
5. Check that facts, author stance, and platform intent still match the source.

## Residual issue checklist

- Generic transition still visible.
- Correct but empty sentence remains.
- Paragraphs have the same shape.
- Reader belief is invented.
- Ending sounds like customer service.
- The rewrite added details the user did not provide.
- The author's original attitude was erased.
- CTA promises too much or sounds like hard selling.
- Platform version no longer sounds native.

## Stop rule

Stop when the remaining issues are small style preferences, not AI-flavor problems. Over-editing is also a quality failure.

## Output

When an audit artifact is needed, record:

- residual issues found
- fixes applied
- facts preserved
- invented details check
- final publishability decision
